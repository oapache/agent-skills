# Abstraction

Use esta skill quando a tarefa for complexa, confusa ou com muitos detalhes.

## Objetivo
Reduzir complexidade sem perder o essencial.

## Processo

1. Identificar o núcleo do problema
- o que realmente importa?

2. Remover detalhes irrelevantes
- o que não impacta a solução?

3. Generalizar
- isso se parece com algo conhecido?

4. Criar modelo simples
- representar o problema de forma clara

## Regras

- simplificar sem distorcer
- manter o essencial
- evitar excesso de detalhe

## Regra suprema

Se está difícil de entender, simplifique.
