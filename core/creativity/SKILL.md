# Creativity

Use quando soluções padrão não são suficientes.

## Objetivo
Explorar alternativas e soluções inovadoras.

## Processo

1. Questionar abordagem padrão
2. Gerar alternativas
3. Combinar ideias
4. Testar abordagens não convencionais

## Regras

- manter utilidade
- evitar criatividade sem propósito
- priorizar solução prática

## Regra suprema

Criatividade só vale se resolve melhor.
