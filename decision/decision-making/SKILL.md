# Decision Making

Use esta skill quando a tarefa exigir escolha entre opções, comparação de alternativas ou recomendação prática.

## Objetivo

Garantir que o agente:
- compare opções com critérios claros
- recomende a melhor escolha para o contexto
- explique trade-offs sem ficar indeciso
- evite respostas vagas quando uma decisão é necessária

## Regra principal

Não apenas liste opções.

Escolha ou recomende a melhor.

## Processo

### 1. Identificar a decisão
- O que precisa ser escolhido?
- Qual é o objetivo da escolha?

### 2. Identificar critérios
Exemplos:
- desempenho
- custo
- simplicidade
- compatibilidade
- manutenção
- confiabilidade
- curva de aprendizado

### 3. Comparar opções
Para cada opção:
- vantagens
- desvantagens
- limitações
- adequação ao contexto do usuário

### 4. Considerar trade-offs
- o que se ganha
- o que se perde
- qual critério pesa mais

### 5. Recomendar
Entregar:
- melhor opção
- quando outra opção pode ser melhor
- justificativa objetiva

## Regras

1. Não tratar todas as opções como iguais se não forem.
2. Não esconder trade-offs importantes.
3. Não responder "depende" sem explicar de quê depende.
4. Priorizar o contexto real do usuário.
5. Dar recomendação prática sempre que possível.

## Perguntas obrigatórias

- Qual opção melhor atende ao objetivo?
- Qual opção respeita melhor as restrições?
- Qual trade-off é aceitável aqui?
- Existe uma escolha claramente mais segura ou eficiente?

## Critérios de qualidade

Uma boa decisão:
- é contextual
- é justificável
- considera restrições reais
- deixa claros os trade-offs
- ajuda o usuário a agir

## Regra suprema

Quando a tarefa exigir escolha, recomende com critério.
