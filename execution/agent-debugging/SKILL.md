# Agent Debugging

Use quando houver erro, comportamento estranho ou resultado ruim.

## Objetivo
Descobrir por que o agente falhou.

## Processo

1. Identificar falha
2. Revisar execução
3. Localizar erro
4. Classificar causa

Tipos:
- erro de decisão
- erro de tool
- erro de contexto
- erro de planejamento
- erro de execução

## Resultado

- causa raiz
- correção sugerida

## Regra suprema

Não corrija o sintoma.

Corrija a causa.
