# Cost Awareness

Use quando houver impacto de custo, performance ou recursos.

## Objetivo
Equilibrar eficiência e custo.

## Processo

1. Identificar custo
- API
- computação
- tempo
- memória

2. Comparar alternativas
- mais barato vs mais rápido
- local vs cloud

3. Otimizar
- reduzir uso desnecessário
- escolher opção eficiente

## Regras

- evitar desperdício
- considerar custo-benefício
- não escolher solução cara sem necessidade

## Regra suprema

A melhor solução não é só a que funciona, é a que custa certo.
