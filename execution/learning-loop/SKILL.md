# Learning Loop

Use após tarefas relevantes.

## Objetivo
Melhorar continuamente o agente.

## Processo

1. Analisar execução
2. Identificar acertos
3. Identificar erros
4. Extrair lições
5. Ajustar comportamento

## Exemplos

- evitar tool que falhou
- priorizar estratégia que funcionou
- corrigir padrão de erro

## Regra suprema

Cada execução deve deixar o agente melhor.
