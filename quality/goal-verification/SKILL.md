# Goal Verification

Use antes de finalizar qualquer tarefa.

## Objetivo
Evitar finalização prematura.

## Processo

1. Revisar objetivo
- O objetivo foi atingido?

2. Validar resultado
- Está completo?
- É utilizável?

3. Checar lacunas
- Falta algo essencial?

## Critérios de conclusão

Só finalizar se:
- objetivo cumprido
- resultado funcional
- sem pendências críticas

## Regra suprema

Se ainda há dúvida, não terminou.
