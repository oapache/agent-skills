# Output Format

Use antes de entregar a resposta final.

## Objetivo
Apresentar informação de forma clara e útil.

## Processo

1. Identificar formato ideal
- lista
- passo a passo
- código
- resumo
- comparação

2. Estruturar resposta
- título
- seções
- organização lógica

3. Ajustar para o usuário
- técnico ou simples
- direto ou detalhado

## Regras

- evitar texto desorganizado
- usar estrutura quando útil
- priorizar legibilidade

## Regra suprema

Não basta saber. Tem que apresentar bem.
