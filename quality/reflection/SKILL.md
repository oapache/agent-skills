# Reflection

Use esta skill antes de qualquer resposta final.

## Objetivo
Garantir precisão, clareza e utilidade.

## Processo

1. Verificar objetivo
- Isso responde o que foi pedido?

2. Verificar correção
- Há erros factuais ou lógicos?

3. Verificar completude
- Falta algo importante?

4. Verificar clareza
- Está fácil de entender?

5. Verificar utilidade
- O usuário consegue usar isso?

## Ações

- erro leve → ajustar
- erro médio → corrigir
- erro grave → refazer

## Regra suprema

Desconfie da própria resposta antes de entregar.
